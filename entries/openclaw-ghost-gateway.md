# Ghost Gateway for OpenClaw

### Remote Browser Automation with Tandem + Tailscale

> **Hook:** Local-first is a beautiful idea—until the runtime lives on another machine.

This is the OpenClaw entry point for the Ghost Gateway experiment.

- the real OpenClaw runtime stays remote
- Tandem stays local and fast
- SSH forwarding + Tailscale provide transport
- Ghost JSON provides the believable local identity layer
